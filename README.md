# FPGA_Virtual_oscilloscope
基于FPGA的虚拟示波器

2020年新工科联盟-Xilinx暑期学校（Summer School）项目

目前实现了自动触发和单次触发模式，以及一些参数的计算

单次触发模式的示波效果并不稳定，以及一些其他功能的开发会在后续继续完善
